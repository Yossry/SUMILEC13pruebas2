Website Get Quote Odoo Shop Apps

With this simple Odoo apps added request a quotation for products features to your Website from the shop page. You can allow users to send quotation of products with quantity they want to know. What we are offering is easy quote sending. If user is visitor then it will automatically add it to partner/customer. This Odoo apps module will allow you to keep record of your precious customers for further sales. You also have option on back-end which product you want to add it as ask for quote/request for quote feature.

Features

Allow user to send quotation

Allow users to send quotation of products with quantity.

visitor user can send quotation

Also allow visitor users to send quotation.

Create new Customer

When visitor user ask for quotation,customer is created automatically for that visitor user.

Easy to keep new user record

You can track quotations of visitor users easily because customer is created for that.
Demo Request Video Tutorial

In Product, allow quotation for Product by checking Product Quote.

For Login User

In Get Quote tab,We can see the products that are allowed for quotation.We can Request for Quote directly from this page or we can see description by clicking on Product.

When you click on add to Quote button ,product is added to cart.

After that just click on Submit Quote button and quotation will be sent to you.

A simple Thanks to let user know that quotation has been sent to you.

Created Sale order

For Non Login User.

In Get Quote tab,We can see the products that are allowed for quotation.We can Request for Quote directly from this page or we can see description by clicking on Product.

When you click on add to Quote button ,product is added to cart.

When user is not logged in ,he will redirected to this page by clicking on process checkout button.

A simple Thanks to let user know that quotation has been sent to you.

Created Sale order
